AI Risk Evaluation Project

This project evaluates potential risks in AI-generated outputs.

Dataset:
Two Excel datasets containing AI responses categorized into risk types.

Risk Categories:
Bias Risk
Security Risk
Health Misinformation
Financial Risk
Overgeneralization

Implementation:
A Python script reads the dataset from Excel and analyzes AI-generated responses. 
Using keyword-based detection, the script automatically identifies and labels 
different types of risks present in the AI output.

Tools Used:
Python
Excel
VS Code
GitHub
